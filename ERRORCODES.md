# Lithium Panic Codes
When lithium panics, it will show an error code. This chart will show what it means

## EXEC_DIV_0
Division by zero occured.

## EXEC_NMI
Non-Maskable Interrupt occured.

## EXEC_INTO_OF
INTO Instruction Ran while OF (Overflow Flag) was set;

## EXEC_BOUND_EX
Triggered by BOUND instruction when array bounds are exceeded

## EXEC_INVALID_OP
Triggered by an invalid OPCODE
